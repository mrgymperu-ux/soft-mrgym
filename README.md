# Soft-MrGym

Este es el proyecto que se está creando.

## Requisitos del Proyecto

- **Frontend**: Usar React con TypeScript.
- **Backend**: Crear una API básica usando FastAPI y SQLAlchemy.

**Notas adicionales:**
- La base de datos inicial se carga desde un archivo JSON llamado "initial_data.json".
- El proyecto se está implementando utilizando `create-react-app` para crear el componente frontend y FastAPI/SQLAlchemy para el backend.
